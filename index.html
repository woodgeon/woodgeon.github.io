About

빠르게 만들되, 운영까지 생각합니다.

저는 백엔드 구조를 먼저 안정적으로 세우고, 사용자가 실제로 체감하는 기능을 서비스 안으로 끌어오는 일에 흥미를 느낍니다. 인증, 데이터 흐름, 파일 처리, 외부 AI API, 배포 파이프라인처럼 서로 다른 요소가 하나의 서비스로 이어지는 지점을 좋아합니다.

프로젝트에서는 Spring Security와 JWT 기반 인증, Docker Compose 환경 구성, GitHub Actions를 활용한 CI/CD, Nginx Reverse Proxy와 HTTPS 설정까지 서비스가 계속 개선될 수 있는 기반을 경험했습니다.
